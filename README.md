# movies-explorer-api
API для приложения по поиску фильмов. Это дипломный проект для Яндекс.Практикума по направлению "веб-разработчик".

## Запуск проекта
Запуск сервера:

    npm run start
Запуск сервера с hot-reload:

    npm run dev
